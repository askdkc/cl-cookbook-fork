---
title: Windows または Mac で Emacs を設定する
---

Emacs は、ほとんどの CL 開発者に好まれている Lisp ソースコードエディタです。

簡単に始めたい場合は、**ポータブル**で**マルチプラットフォーム**な CL 開発環境である [Portacle](https://shinmera.github.io/portacle/) を入手してください。含まれているものは次のとおりです。

- Emacs、少しカスタマイズ済み
- Slime
- SBCL
- Quicklisp
- Git

Portacle をダウンロードし、ダブルクリックして実行するだけです。
